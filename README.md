# Usage

dotnet DatabaseVisualizer --connectionString "my-connection-string" --includeSchemas "" --excludeSchemas "" --excludeColumns "" --render "classDiagram;erDiagram" --output "classDiagram.mmd;erDiagram.mmd"

# Arguments

## connectionString [cs]
## includeSchemas [is]
## excludeSchemas [es]
## excludeColumns [ex]
## render [r]
## output [o]
